# CSS Notes

## What is CSS?

CSS stands for Cascading Style Sheets.

Purpose:
Styles HTML elements.

## Class Selector

.card {}

Purpose:
Styles elements having class="card".

## display: flex

Purpose:
Turns an element into a Flexbox container.

## flex-direction

Values:
- row
- column

Purpose:
Controls direction of items.

## align-items:
- center
- flex-start
- flex-end

Purpose:
Align items horizontally.

## justify-content

Common values:
- center
- space-between
- space-around

Purpose:
Aligns items on the main axis.

## Padding

Purpose:
Space inside an element.

Example:

padding: 20px;

## Margin

Purpose:
Space outside an element.

Example:

margin: 20px;

## Border

Example:

border: 1px solid black;

Purpose:
Creates an outline.

## Border Radius

Example:

border-radius: 10px;

Special:

border-radius: 50%;

Creates a circle.

## Background Color

background-color: white;

Purpose:
Changes background color.

## Box Shadow

box-shadow: 0 4px 8px rgba(0,0,0,0.1);

Purpose:
Adds shadow.

## Hover

button:hover {}

Purpose:
Applies styles when the mouse is over an element.

## Transition

Purpose:
Animates property changes smoothly.

## Cursor

cursor:pointer;

Purpose:
Changes mouse cursor.

## Font Family

Example

font-family: Arial, sans-serif;

Purpose:
Changes font.
