# HTML NOTEs

# What is HTML?
- HTML stand for Hypertext Markup Language.
- It gives structure to a webpage.

## Basic structure

<!DOCTYPE html>
<html>
  <head>
  </head>
  
  <body>
  </body>
</html>

Purpose:
- html: Root element
- head: Information about the page
- body: Visible content

## Image

Tag: 
<img src="" alt="">

Attributes:
- src: Image location
- alt: Description if Image doesn't load

## Headings

Tags:
- h1 <h1>Arjun</h1>
- h2
- h3
- h4
- h5
- h6

Purpose:
Used for headings.

## Paragraph

Tag:
<p></p>

Purpose:
Used to display text.

## Lists

Unordered List:

Tag:
<ul>
   <li>HTML</li>
</ul>

Ordered List:

Tag:
<ol>
   <li>CSS</li>
</ol>

## Button

Tag:
<button>Click Me</button>

Purpose:
Creates a clickable button.

## Div

Tag:
<div></div>

Purpose:
Groups multiple elements together.

## Class

Example:
<div class="card"></div>

Purpose:
Labels elements so CSS can style them.
